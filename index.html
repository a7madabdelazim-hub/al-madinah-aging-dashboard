⏪ Reverse Month-End Macro

If you accidentally ran the Month-End Macro, or were just testing, click the button below.

  • This will instantly restore all customer balances to exactly how they were BEFORE the macro was run.
  • It will restore your Current Month Invoices and Collections.
  • This action only works for the LAST macro run.


⚠️ Month-End Process

Clicking the button below will execute the following actions:

  • Shift all aging buckets one column to the right.
  • Move current Invoices to the 0-30 bucket.
  • Apply current Returns to the 0-30 bucket.
  • Apply current Collections to the buckets (oldest first).
  • Lock in Current Month Actuals to History.
  • Clear current month holding areas.
